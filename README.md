# Rocket-Launch-Game
Game created with Unity3D and C#

## Instructions
GOAL: To reach the other green landing pad!

CONTROLS: 
- Space - Boost the Rocket
- A - Rotate Left
- D - Rotate Right
- Esc - Exit Game

## How To Play
- Clone the repo and run "Builds/Rocket Launch.exe"
